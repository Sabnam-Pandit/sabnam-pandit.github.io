Apartment for rent classification

Website Link to dataset

This data set is useful in training machine learning algorithms for different task like Classification,Clustering and Regression. By using the features in the datasets like number of bedrooms,number of bathrooms , address,city name ,state etc. a machine learning regression model can be trained to predict the price of an apartment. Similarly,Using the features in the datasets like amenities,has_photo,fee etc. a machine learning classification model can be trained to classify the better quality apartment.

Author:David Bollier,Rapporteur

Institute:The Aspen Institute

The book is a smooth read for a beginner.It provides an overview of the big data world to the reader while explaining how data is rapidly generated in massive amounts. The author explains the sources that aids in generation in big data.With the advancement in new tech,cloud computing data is generated in humongous amount.It also address the uses and creation of data in different domain like health,geolocation,social network etc.What I find interesting about the book is that it gives a different perspective on the impact of data science in real world.In addition to the introduction to the big data world,it also highlights the social and ethical implications of big data,data abuse and the challenges like violation of an individual’s privacy.
